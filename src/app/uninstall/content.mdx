# **How to Uninstall**
We're sorry Termbook didn't work out for you! Here is how to remove Termbook.

## **Remove the executable command from your terminal setup file**
In your terminal setup file (.zshrc, .bashrc, .bash_profile, profile.ps1, etc.) find the command you inserted during setup that contains the file termbook_yourplatoformhere and delete it.

## **Remove the executable file from you download location**
Find the location where you originally downloaded the file and delete it from your filesystem.

**We are sorry to see you go!**
